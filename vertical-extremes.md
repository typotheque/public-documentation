# Vertical extremes in common texts

Devanagari देवनागरी:

> ड्डू \
> कँकॅंर्किं

Gujarati ગુજરાતી:

> ડ્ડૂ \
> કૅંર્કિં

Gurmukhi ਗੁਰਮੁਖੀ:

> ਕ੍ਹੂ \
> ਕੇਂੴ

Bangla বাংলা:

> ক্ট্রট্রূ \
> ঐর্কিকিঁ

Odia ଓଡ଼ିଆ:

> ତ୍ଥୂ \
> କୈର୍କିଁ

Telugu తెలుగు:

> క్టైక్ఖై \
> కీఖో

Kannada ಕನ್ನಡ:

> ಷ್ಟ್ರ \
> ಣಿ

Malayalam മലയാളം:

> Simplified orthography \
> ക്ലസ്സ \
> കീ

> Traditional orthography \
> ക്യൂസ്റ്റ്രൂസ്കൂ \
> കീൎക

Tamil தமிழ்:

> ஸ்ரீணு \
> கீ

Sinhala සිංහල:

> බ්‍රූ \
> ඪි

Meetei ꯃꯤꯇꯩ:

> ꯀ꯭ꯔꯨ \
> ꯃꯩ

Ol Chiki ᱚᱞ ᱪᱤᱠᱤ:

> ᱚ
> ᱚ

- This script doesn’t have ascending/descending structures.
